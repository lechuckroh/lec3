# LEC3

## Requirements
* Go v1.4+
* [gb](https://getgb.io)

## Build
To build all:
```bash
$ gb build all
```

To build specific executable:
```bash
$ gb build [packageName]
```

Available package names are:
* `lec-conv`
* `lec-ip`

After successful build, you can find executable files under `./bin` directory.
